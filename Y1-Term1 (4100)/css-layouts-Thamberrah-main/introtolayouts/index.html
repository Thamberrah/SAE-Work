Today we'll be looking at ways to layout our our HTML in a consistent manner, but first, a little history.

CSS 1

In CSS 1, the only way to structure content was through using the table tag. While this allowed sites to be consistent, using this tag made sites slower to load.

CSS 2

CSS 2 allowed the use of float:left and float:right. This allowed more flexibility in layouts, but still made designing to multiple screens challenging. Most sites now required both a desktop and a mobile version. Any animations required heavy use of Javascript

CSS 3

This version of CSS gave us a number of tools we can use today: ems, media queries, grid and flex layouts, and animations
