Here's a list and description of various design patterns the students have researched

BEM-Block Element Modifier

This is a tool for naming things.

Block- the outermost element

Element- children of the block

Modifier- A flag to change the behaviour

Atomic Design

Inspired by chemistry, breaks down website into atoms or key components.
  • Atoms- ex. field or input
  • Molecules-picture+caption
  • Organism-header
  • Templates-Landing Page
  • Pages- Home page
